Predicting the Trend on Stock Market Using Machine Learning with Python

  • Data Source & Collection – Stock market data was pulled via the yfinance API for Apple (AAPL) as the primary stock, plus GE, Google, IBM, and Microsoft for correlation analysis, covering 10 years (Jan 2010–Jan 2020) with ~2,515 daily records.

  • Feature Engineering – Four key features were engineered from raw OHLCV data: Adjusted Close Price (target variable), Volume, HL_PCT (intraday volatility), and PCT_change (bullish/bearish daily movement).

  • Machine Learning Models – Two variants of K-Nearest Neighbors (KNN) were implemented: Uniform-weighted KNN (k=5, Euclidean distance) and Distance-weighted KNN, compared against a Moving Average baseline (10-day window).

  • Tools & Technologies – Built entirely in Python, using scikit-learn for model training, Standard Scaling for preprocessing, matplotlib for visualization, and Tkinter for a 7-module GUI application (data download, correlation analysis, preprocessing, model training, prediction, and visualization).

  • Evaluation & Results – Models were evaluated using the R² score, with Distance-weighted KNN achieving the best performance at 98.12%, outperforming Uniform KNN (96.54%) and the Moving Average baseline (89.3%).
